Hello, welcome to my intake game Falling Colors.

The objective is very simple, collect as many colors as possible.
The game gets progressively harder based on how long you survived.
When the falling colors are past your player you will receive damage.

Good luck!

Controls: A and D

Intergrated Developement Environment:
recommended->visual studio 2022 however a older visual studio is also compatible

Steps:
1-> Extract the file wherever you like.
2-> Open the Falling Colors.sln file
3-> Make sure in your IDE to select x86
4-> Press the green arrow to play the game or use the shortcut Ctrl + F5
5-> Enjoy
